# AI Rater Portfolio

This repository showcases practical examples of **search quality evaluation**, **AI response grading**, and **data labeling**—the kind of work commonly performed in AI Rater / Search Evaluator roles (e.g., Appen, TELUS AI, Outlier, Remotasks).

## What’s inside

- **Search Quality Evaluation**
  - Rated SERP-style examples with intent analysis and clear rationales.
  - A reusable template for creating new ratings.
- **AI Response Grading**
  - Scored examples for LLM outputs (accuracy, safety, usefulness).
  - A reusable grading rubric + template.
- **Dataset Annotation**
  - Small labeled dataset sample (CSV) + annotation guidance.
- **Guidelines Study Notes**
  - Notes + checklists that mirror common rater guideline concepts.

## How to use

1. Clone or download this repo.
2. Replace the example files with **your own** examples (keep the structure).
3. Add 2–5 new examples per section over time.
4. Pin this repo to your GitHub profile.

## Suggested next upgrades

- Add a small notebook (optional) that summarizes labels or computes inter-annotator agreement.
- Add a “case-study” page describing your evaluation workflow in 1–2 pages.
