# Quality Checklist (Quick)

Use this as a quick pass before finalizing a rating.

## Intent
- Did I identify the likely user intent correctly?
- Are there constraints (device, location, urgency) affecting intent?

## Relevance
- Does the result directly answer the query?
- Is it on-topic but unhelpful (then “slightly”)?

## Trust
- Is the source authoritative for the topic?
- Any red flags (misleading claims, spam, unsafe advice)?

## Helpfulness
- Can the user complete the task with this result?
- Is information up-to-date enough for the query?

## Consistency
- Do my ratings match my rationales?
- Would another rater reading my notes agree?
