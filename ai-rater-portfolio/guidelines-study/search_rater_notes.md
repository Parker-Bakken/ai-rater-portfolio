# Search Rater Notes (Study Cheatsheet)

This cheatsheet summarizes common rater guideline concepts.

## Intent Types
- Informational (learn)
- Navigational (go to a specific site/page)
- Transactional (buy/compare)
- Help / Troubleshooting (fix a problem)
- Local (near me / specific area)

## Relevance / Needs Met (conceptual)
- **Fully / Highly Meets:** directly satisfies the user’s need
- **Moderately Meets:** useful, but missing something
- **Slightly Meets:** related but not very helpful
- **Fails:** irrelevant, misleading, or low value

## Page Quality Signals (conceptual)
- Clear authorship / credibility
- Updated, accurate content
- Useful structure (headings, steps, summaries)
- Lack of deception (clickbait, spammy ads, unsafe downloads)

## Common Failure Modes
- Misread intent (answers a different question)
- Thin content / listicles with no substance
- Untrustworthy sources for high-stakes queries (medical, legal, finance)
